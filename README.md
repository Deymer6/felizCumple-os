# Digital Celebration - Sorpresa de Cumpleaños 

![Status](https://img.shields.io/badge/Status-Live-brightgreen)
![Fun](https://img.shields.io/badge/Experience-Interactive-ff69b4)

Un proyecto interactivo y multimedia diseñado para crear una experiencia de celebración única. Este repositorio contiene una serie de páginas animadas y una selección musical cuidadosamente elegida para acompañar cada momento de la sorpresa.

---

## ✨ Características de la Experiencia

El proyecto no es solo una página, sino un recorrido interactivo:
* **💌 Carta Animada:** Un detalle especial con `cartaAnimada.html`.
* **🎁 Sorpresa Final:** Un cierre emocionante con `sorpresaFinal.html`.
* **🎶 Ambiente Musical:** Integración de múltiples pistas para diferentes estados de ánimo (desde `pianomoment.mp3` hasta `pop-beat`).
* **🎈 Interactividad:** Efectos de sonido como `balloon-pop` para una experiencia inmersiva.

---

## 🎵 Playlist Incluida

El proyecto cuenta con una amplia variedad de recursos auditivos:
* **Relajantes:** `pianomoment.mp3`, `moonlightcoffee.mp3`.
* **Alegres:** `sunny.mp3`, `smile.mp3`, `funnysong.mp3`.
* **Instrumentales:** `ukulele.mp3`, `moose.mp3`.
